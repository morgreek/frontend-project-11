# RSS aggregator (third Hexlet project)

### Info
Simple application for reading RSS feeds.  
``Node.js v18.15.0``

### Installation
1. Cloning repository:
```shell
git clone https://github.com/morgreek/frontend-project-11.git
```

2. Change directory:
```shell
cd frontend-project-11
```

3. Install app:
```shell
make install
```
